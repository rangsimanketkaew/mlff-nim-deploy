# Note for Installing Required Libraries

## NVIDIA TensorRT

- Build TensorRT C++ from source
Ref: https://github.com/nvidia/tensorrt#build

- Install TensorRT Python via Pip
Ref: https://docs.nvidia.com/deeplearning/tensorrt/latest/installing-tensorrt/install-pip.html
| This will be used as backend for pytorch compile

- Get started: build your first engine
Ref: https://docs.nvidia.com/deeplearning/tensorrt/latest/getting-started/build-your-first-engine.html
